# Travel Navigation Macropad

## Overview

This project is a compact 4-key travel-themed macropad powered by the Seeed Studio XIAO RP2040. It features four directional navigation keys and two SK6812 MINI-E RGB LEDs.

## Features

* Seeed Studio XIAO RP2040
* 4 Mechanical Switches
* 2 SK6812 MINI-E RGB LEDs
* Compact 2-Layer PCB
* Custom Designed Case

## Key Layout

* **8** → Up Arrow
* **6** → Right Arrow
* **4** → Left Arrow
* **2** → Down Arrow

The layout is inspired by classic numpad navigation controls and matches the travel theme probably.
